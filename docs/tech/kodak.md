# Kodak Brand Analysis (hilightbrands-kodak.co.kr)

## 1. 디자인 정체성 (Design Identity)
- **핵심 키워드:** Heritage, Cinematic, Bold, Analog-Digital Hybrid.
- **색상 팔레트:**
    - **Primary Yellow:** `#FFD200` (전통적인 코닥 옐로우)
    - **Accent Red:** `#E10019`
    - **Base:** `#000000` (Black), `#FFFFFF` (White)
    - **Neutral:** `#F4F4F4` (Light Gray)
- **타이포그래피:**
    - **Headings:** 고딕 계열의 굵은 볼드체 (Impact 있는 Sans-serif)
    - **Body:** 가독성이 좋은 본고딕(Noto Sans KR) 또는 유사 스타일.

## 2. 레이아웃 및 UX 패턴
- **헤더 (Header):** 
    - 상단 고정(Sticky) 형태.
    - 투명 배경에서 스크롤 시 반전되거나 배경이 생기는 방식.
    - 로고 중앙 배치 또는 좌측 배치.
- **히어로 섹션 (Hero Section):**
    - 풀 스크롤 영상 또는 고화질 시네마틱 이미지.
    - 텍스트 오버레이와 강력한 Call-to-Action(CTA) 버튼.
- **컴포넌트 특징:**
    - 필름 롤을 연상시키는 가로 스크롤 레이아웃.
    - 격자(Grid) 기반의 제품 노출, 여백(Whitespace)을 충분히 활용한 잡지 스타일.
    - 마우스 호버 시 이미지 확대 또는 색상 반전 효과.

## 3. WISE 템플릿 적용 전략
- **구조:** `minimalist-studio`의 정갈한 구조를 유지하되, 코닥 특유의 브랜드 컬러와 시네마틱한 대형 미디어 섹션을 강화.
- **컴포넌트:** 
    - `FilmGrid`: 필름 스타일의 커스텀 그리드.
    - `CinematicHero`: 비디오 배경의 히어로 섹션.
    - `HeritageStory`: 브랜드 역사를 보여주는 타임라인 또는 잡지 스타일 섹션.
