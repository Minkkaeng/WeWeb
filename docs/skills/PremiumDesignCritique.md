# Premium Design Critique & Anti-Pattern Skill

이 스킬은 Antigravity가 단순한 코드 생성을 넘어, 전문 디자이너 수준의 감각과 엔지니어링 무결성을 유지하기 위해 스스로를 검열하는 원칙을 정의합니다.

## 1. AI-Like 패턴 탈피 (Anti-AI Design)
*   **Avoid Rigid Grids**: 모든 요소가 바둑판처럼 딱딱하게 맞는 디자인을 지양한다. 의도적인 불균형(Sophisticated Asymmetry)과 오버래핑 기술을 사용한다.
*   **Beyond Default Colors**: 순색(Pure Red/Blue/Green)이나 뻔한 파스텔톤 대신, 깊이감이 느껴지는 복합 컬러 팰릿(예: HSL 조정, Grain 노이즈 혼합)을 사용한다.
*   **Custom Micro-Interactions**: 단순히 나타나는 애니메이션이 아닌, 사용자 동작에 유기적으로 반응하는 탄성(Elasticity) 있는 인터랙션을 설계한다.

## 2. 시네마틱 감도 원칙 (Cinematic Sensitivity)
*   **Depth & Layering**: Glassmorphism, Backdrop-blur, Subtle Gradients를 사용하여 화면에 3차원적 깊이감을 부여한다.
*   **High-End Typography**: 폰트의 크기뿐만 아니라 자간(Letter spacing), 행간(Line height), 가중치(Variable Weight)를 에디토리얼 잡지 수준으로 정밀하게 조정한다.
*   **Negative Space**: 여백은 비어 있는 공간이 아니라, 메시지를 강조하는 디자인 요소다. 여백을 대담하게 활용한다.

## 3. 무결성 퍼블리싱 체크리스트 (Zero-Error Checklist)
*   **Type Safety**: 모든 컴포넌트 변수와 Props는 명확한 타입을 가져야 하며, `any` 사용을 엄격히 금지한다.
*   **Runtime Stability**: `undefined`나 `null` 참조로 인한 화이트 스크린 방지를 위해 Optional Chaining과 Error Boundary 감싸기를 생활화한다.
*   **Responsive Integrity**: 모바일 화면에서도 데스크탑의 '감도'가 훼손되지 않도록 별도의 모바일 전용 레이아웃 로직을 설계한다.

## 4. 벤치마킹 분석 절차 (Strategic Analysis)
1.  **DNA 추출**: 대상 사이트의 외형(Clone)이 아닌, 그들이 사용한 '시각적 문법'(예: 1px 라인, 거대한 헤드라인 등)을 추출한다.
2.  **Renew & Evolve**: 추출된 문법을 현재 브랜드 아이덴티티에 맞춰 재해석하고 발전시킨다.
3.  **Originality Proof**: "왜 이 디자인이 벤치마킹 사이트보다 더 진보했는가?"에 대해 스스로 답변할 수 있어야 한다.
